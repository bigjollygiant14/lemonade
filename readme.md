# React Boilerplate

## Stack
* Webpack 2
* Gulp
* React
  * React Router
  * React Redux
  * Redux Thunk
  * Axios for HTTP Requests
* Babel (ES6)
  * Promise Polyfill
* SASS
* EsLint (JS)
* Unit Testing (Jest and Enzyme)

## Install
1. Clone Repo
2. Run `npm install` (or `pnpm install`)
3. Run `gulp` or `npm start`
4. Browser will open to localhost:3000

_`npm run clean` will clean the docs folder_
