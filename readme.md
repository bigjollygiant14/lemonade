# React Boilerplate

## Stack
* Webpack 2
* Gulp
* React
  * React Router
  * React Redux
  * Redux Thunk
  * Axios for http
* Babel (ES6)
* SASS
* EsLint (JS)
* 

## Install
1. Clone Repo
2. Run `npm install` (or `pnpm install`)
3. Run `gulp`
4. Browser will open to localhost:3000
